# Rapidoc

Generate REST API doc.

It is based in restapi_doc: https://github.com/Alayho/restapi_doc

## Installation

Add this line to your application's Gemfile:

    gem 'rapidoc'

And then execute:

  $ bundle

Or install it yourself as:

  $ gem install rapidoc

## Usage

  rake rapidoc:generate
